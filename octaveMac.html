Installing Octave for MacOS X

The easiest way to install Octave in Windows is through Macports, an interface for a large collection of open source programs. Each program in the collection is packaged for easy installation and deinstallation. Macports has packages for mathematical tools like Maxima, R, LyX, LaTeX, and many generally useful tools, but we will stick to the minimum needed to run Octave.
