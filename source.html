Tea Time Numerical Analysis Experiences in Mathematics

Source

The source for this project includes LyX, Octave, GeoGebra, wxMaxima, xfig, text, and graphics files. Minimally, you will need a LyX installation to view and edit the textbook source files. Other tools are required to view and edit the diagram source files. Here are several ways you can get the source.

  • Get a tarball (.tar.gz file).
  • Get a zipball (.zip file).
  • git clone https://github.com/lqbrin/tea-time-numerical.git
  • See this project at github.

As are the textbook and Octave, all the tools used to create this textbook are open source projects. You may download and use them free of charge, and even study and modify their source code if you wish. Besides the software highlighted above, GIMP and the command line utility pdfcrop were also used. Everything needed is neatly packaged in Mathbuntu, a complete operating system that can be run in a virtual machine.
